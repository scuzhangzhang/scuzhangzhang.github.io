ubuntu 20.04 安装 opensdk 11

发表于 2022-08-07 分类于 系统基础

踩坑过程

sudo apt-get update
sudo apt-get install openjdk-11-jdk
都是扯淡，还有说是因为没有添加jdk 仓库
sudo apt add-apt-repository ppa:openjdk-r/ppa
添加了还是不得行，最后还是自己下jdk 才得行
wget https://download.java.net/java/GA/jdk11/9/GPL/openjdk-11.0.2_linux-x64_bin.tar.gz

下载完成后执行了以下命令
sudo mkdir /usr/local/jdk
sudo chown ubuntu:ubuntu /usr/local/jdk
tar xzvf openjdk-11.0.2_linux-x64_bin.tar.gz
cp -r openjdk-11.0.2 /usr/local/jdk/

接下来配置环境变量

sudo vim /etc/profile
添加
export JAVA_HOME=/usr/local/jdk/jdk-11.0.2
export PATH= ${JAVA_HOME}/bin:$PATH

请务必记住，不要在=后面加空格，不然会造成无法登陆，运行source /etc/profile 会提示
-bash: export: `xxxx’: not a valid identifier

最后wq保存后运行 souce /etc/profile 即可。
注销后重新登陆，运行java javac java –version 无异常就是安装成功了。
