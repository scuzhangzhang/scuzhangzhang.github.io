Docker环境

发表于 2023-07-13 分类于 web

1、Docker基本概念

一个非常轻量级的虚拟环境，没有独立IP，依赖于宿主机。 Docker之间可以互相通信。外部设备无法访问Docker，必须把端口映射给宿主机。

注意事项：

  • 尽量让一个容器做一件事情
  • 尽量使用挂载的方式把数据文件挂载到容器中
  • 尽量不要使用交互环境来直接操作容器，而是在宿主机上执行命令。
