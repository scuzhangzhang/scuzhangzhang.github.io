Linux 基础

1、Centos 7 软件安装

  • rpm 安装

    • rpm -ivh 软件包
    • rpm -e 软件包
    • rpm -qa |grep 软件包

  • 源码安装

    • 寻找configure setup.sh install.sh

      • ./configure
      • make
      • make install

      发现缺少依赖，但是系统已经安装对应的软件，可以尝试安装一下-devel,如pcre-devel

  • yum 安装

    • yum list
    • yum install
    • yum repolist 列出当前的源
    • yum deplist 查看依赖库
    • yum erase 软件包

2、相关命令

net-tools 以及iproute2 两套网络相关命令体系

  • ip addr add 192.168.101.111/24 dev ens33

    ifconfig ens33 192.168.101.111

    临时增加一个IP地址，一个网卡可以绑定多个IP地址

  • ss -anl

    netstat -anlop 列出当前端口占用情况

  • traceroute 跟踪路由

  • curl -O url

  • wget -O xxx URL

  • tcpdump

    • tcpdump tcp and dst port 80 -i ens33 -w xx.pcap

3、Linux 命令

  • tee 命令同时打印到控制台以及文件中

  • 1>file 2>file

    1 代表正确的重定向，2代表错误的重定向

  • 关于参数的判断语句

    注意[] 前后的空格 $ 必须加引号

    关于判断文件以及目录的参数

  • 引号

    • 双引号 变量会被引用

    • 单引号，变量不会被引用

    • 反引号，把里面的内容当作一个命令

  • $? 非零代表上一条命令失败

  • $0 代表脚本的名称，$1 是第一个参数

  • $@ 和$* 返回脚本的参数，前者是多个字符串，或者是一个字符串

  • expr 数字命令 必须有空格，不能处理小数

  • 逻辑运算

    • a 与

    • -o 或

    • ! 非

  • 字符串

    • -z 为空字符串

    • -n 为非空字符串

  • 数值比较

    • -eq   数值相等

-ne  数值不等

-le  小于等于

-ge 大于等于

-gt 大于

-lt 小于

  • 函数

    关于括号的一些用法

    • (()) 双圆括号，用于运算，可以替代expr，数学运算

    • ${} 可以去特定参数或变量的值 ${1}

    • $() 可以代替反引号执行圆括号中的命令

    • $[] 用于整数运算，也可以代替expr
