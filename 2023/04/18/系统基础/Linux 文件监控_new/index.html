Linux文件监控小脚本

1、通过md5sum 对比

2、通过diff 命令比较内容的修改，目录里面文件的新增也可以检测到

修改了xxx

./origin/anaconda-ks.cfg: OK
./origin/monitor.sh: OK
./origin/test.txt: OK
./origin/xxx: FAILED
md5sum: WARNING: 1 computed checksum did NOT match

删除了xxx

./origin/anaconda-ks.cfg: OK
./origin/monitor.sh: OK
./origin/test.txt: OK
md5sum: ./origin/xxx: No such file or directory
./origin/xxx: FAILED open or read
md5sum: WARNING: 1 listed file could not be read

新增

[root@192 ~]# diff origin after
Only in origin: xxx1
